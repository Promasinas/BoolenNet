# v0.3 — Chatbot 变体集合

**日期**: 2026-06-06 | **大小**: 60–85KB

## 包含程序

| 程序 | 功能 |
|------|------|
| `chatbot_byte.exe` | 字节流处理版 |
| `chatbot_compute.exe` | 计算模式版 |
| `chatbot_full.exe` | 完整训练流水线 (早停+检查点+导出) |
| `chatbot_trained.exe` | 预训练对话模型 |
| `train_long.exe` | 长训练测试程序 |

## 运行

```bash
./chatbot_trained.exe   # 预训练对话
./chatbot_full.exe      # 完整流水线
```
