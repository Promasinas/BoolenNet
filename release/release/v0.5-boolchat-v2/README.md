# v0.5 — BoolChat v2 (64 Q&A 路由树)

**日期**: 2026-06-07 | **大小**: 62KB | **架构**: 127 节点路由树 + 64 叶子 + N-gram 编码

## 功能

🏆 首个大规模 Q&A 路由树。N-gram 哈希编码 + 深度 6 路由树 + XOR 解码输出。

## 运行

```bash
./boolchat_v2.exe
```

## 架构

```
输入文本 → N-gram Hash → 深度6路由树(127节点)
         → 63内部节点(逐bit决策) → 64叶子节点
         → 每叶子: XOR解码 → 输出回复字节
```

| 编码方式 | N-gram 哈希 |
