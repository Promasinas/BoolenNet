# v0.4 — BoolChat (XOR 路由对话)

**日期**: 2026-06-06 | **大小**: 62KB | **架构**: 纯 XOR 路由 + 字节输出

## 功能

首个纯 XOR 路由对话机器人。输入文本 → N-gram 编码 → XOR 路由 → 字节输出。

## 运行

```bash
./boolchat.exe          # 对话
./boolchat_train.exe    # 训练版
```

| 路由方式 | XOR bit-flip |
| 字节匹配 | ✅ 路由正确 |
