# BoolNet Chatbot Release

## 运行

双击 `run.bat` 或在终端执行 `chatbot.exe`

## 可回答的问题 (16 组 Q&A)

| # | 问题 | 答案概要 |
|---|------|---------|
| 0 | 你好 | 问候 |
| 1 | 你是谁 | 自我介绍 |
| 2 | 天气 | 无法获取实时天气 |
| 3 | 时间 | 查看系统时间 |
| 4 | BoolNet是什么 | 框架介绍 |
| 5 | 怎么训练 | Tsetlin训练说明 |
| 6 | 再见 | 告别 |
| 7 | 谢谢 | 礼貌回应 |
| 8 | 你会做什么 | 能力说明 |
| 9 | 谁创造了你 | 来源说明 |
| 10 | Tsetlin | 自动机解释 |
| 11 | 布尔路由器 | 路由器解释 |
| 12 | 保存模型 | 导出说明 |
| 13 | 加载权重 | 加载说明 |
| 14 | 你的名字 | BoolBot |
| 15 | 帮助 | 命令列表 |

## 性能

- **推理速度**: < 1ms (单个 Router→Memory forward)
- **内存占用**: ~100KB (16 个 16-byte 向量 + 网络参数)
- **网络架构**: Router(128bit) → Memory(16 cells)

## 命令

- 输入中文问题 → 自动匹配回答
- `list` → 显示所有可问的问题
- `quit` → 退出
