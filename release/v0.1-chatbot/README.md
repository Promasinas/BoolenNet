# v0.1 — Chatbot (首个对话机器人)

**日期**: 2026-06-06 | **大小**: 85KB | **架构**: Router(128b)→Memory(4c)

## 功能

16 组内置 Q&A，Tsetlin 训练，级联路由树。

## 运行

```bash
./chatbot.exe
> hello
> list
> quit
```
